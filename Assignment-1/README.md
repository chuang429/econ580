# ECON 580 Git Tutorial Project

This repository contains the completed practice for all 14 chapters of the Software Carpentry Git tutorial.

## Structure
- All tutorial exercise files are located in the `recipes/` folder
- Each chapter's work is documented with clear commit messages

## Chapters Completed
- Episode 4: Tracking Changes (guacamole.md)
- Episode 6: Ignoring Things (.gitignore)
- Episode 7: Remotes in GitHub
- Episode 8: Collaborating (hummus.md)
- Episode 9: Conflicts
- Episode 10: Open Science (open_science.md)
- Episode 11: Licensing (LICENSE)
- Episode 12: Citation (CITATION)
- Episodes 1-3, 5, 13-14: Completed through reading/practice
